SVG ICON PLACEHOLDERS CREATED

The manifest.json references PNG files (icon-16.png, icon-32.png, etc.)
but SVG files were created as placeholders.

FOR TESTING: 
- Chrome accepts SVG icons unofficially
- Change manifest.json extensions from .png to .svg temporarily

FOR PRODUCTION:
- Convert SVG to PNG using:
  * Online: https://cloudconvert.com/svg-to-png
  * CLI: inkscape icon-16.svg -o icon-16.png (if inkscape installed)
  * Or use any graphics editor (GIMP, Photoshop, etc.)

ICON DESIGN:
- Orange background (#FF6B35 - OneForm brand color)
- White "1F" text (OneForm abbreviation)
- Simple, readable at all sizes
